## 🚀 Utilisation

### Méthode 1 — Direct (recommandé)
```bash
# Cloner le repo
git clone https://github.com/Ahlamghribi/landing-page-strategixs-assistance.git

# Ouvrir directement dans le navigateur
# Double-cliquer sur "page 1.html" ou "page2 .html"
```

### Méthode 2 — Déploiement Netlify
1. Aller sur [netlify.com](https://app.netlify.com)
2. **Add new site → Import from Git → GitHub**
3. Sélectionner ce repository
4. **Publish directory** : laisser vide (racine)
5. Cliquer **Deploy** ✅

### Méthode 3 — Drag & Drop Netlify
1. Aller sur [app.netlify.com/drop](https://app.netlify.com/drop)
2. Glisser-déposer le fichier HTML directement
3. ✅ En ligne instantanément
